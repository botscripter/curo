### What is Curo üôã
The main purpose of Curo is to be the enabler of extrem fast development of your digital transformation with Camunda. Curo provides a modern Tasklist written in Angular and many other solutions for your daily Camunda development pains. ü•≥

### Features ‚ú®

#### Authentication / Authorization
- BasicAuth & OAuth 2
- Custom endpoint filter support
- User & Group Federation
- Login Success Interceptors
#### REST-Endpoints
- Better response objects
- FlowToNext
- Menu generation
- Extending filter support
- OpenApi-UI support
- Business Logic Error
#### Others
- Initial data generation
- Variable Helper
- Variable Helper Mock
- Camunda configuration
